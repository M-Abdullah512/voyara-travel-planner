# Voyara Travel Planner ✈

A full-stack travel planner web application built with Django and PostgreSQL.

## Features
- Browse Pakistan and World travel packages
- User Signup and Login
- Book travel packages online
- Passenger Dashboard to track bookings
- Admin Dashboard to manage packages and bookings
- Confirm or Cancel bookings from admin panel
- Google Maps integration
- Beautiful responsive design

## Tech Stack
- **Backend:** Django 6
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** Django built-in Auth

## Pages
- Home Page
- Login & Signup
- Package Detail
- Book Ticket
- My Bookings
- Passenger Dashboard
- Admin Dashboard

## How to Run
```bash
# Activate virtual environment
venv\Scripts\activate

# Install packages
pip install django psycopg2-binary pillow

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
```
